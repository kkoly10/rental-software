# Design reference — mockups

Standalone HTML exports and zip bundles of the **Korent v2 "carnival"** design
mockups, kept for reference while the dashboard is brought in line with them.
These are **not** part of the app build — nothing imports them.

| File | What it is |
| ---- | ---------- |
| `Korent Dashboard v2 (standalone) (1).html` | v2 operator dashboard (current target — icon sidebar, hero, stat strip, copilot). |
| `Korent Dashboard (standalone).html` | earlier dashboard cut. |
| `Korent Storefront Template (standalone).html` | public storefront template. |
| `Patch3 Preview (standalone).html` | Patch 3 preview. |
| `Patch 4.zip` | catalog photo grid + order filter chips patch. |
| `korent.zip`, `korentpatch 1.zip` | earlier patch bundles. |

They're large self-contained exports — moved here to keep the repo root clean.
